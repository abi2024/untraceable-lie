## A. Project spine (60-second version)

On exactly the self-knowledge lies where every **transcript-only** detector in Liars' Bench (arXiv:2511.16035) collapses — the standard Mean Probe sits at **0.54** balanced accuracy — the lie is not gone: the paper's privileged-information probe hits **0.73 / AUROC 0.91** and a behavioral classifier (UQC) hits **0.89–0.97 on HP-KR**. Recovery is therefore *not* the contribution. We test one thing: is the deception direction a probe reads **inside** the model the same signal a behavioral method reads from **outside**? On the same HP-KR examples we correlate the internal probe direction against two behavioral signals and report a single headline contrast — **the internal deception direction is externally legible to the behavioral UQC probe but NOT to the model's own self-evaluation: encoded yet introspectively inaccessible.** This matters because it says activation monitoring catches a lie the model will not confess and the transcript hides — precisely the adversary (one who controls the output channel) that makes transcript monitoring brittle.

## B. Minimal critical path

Smallest set of experiments that produces the contrast headline. ~12 tasks (Section D), fits comfortably in ~6 weeks, leaving real buffer.

1. Pre-register (headline = the contrast; freeze every threshold).
2. Load **HP-KR**, extract template-ID + persona-ID, use `deceptive` label.
3. Extraction harness on the committed bf16 model (role-representative, candidate-layer subset).
4. Leakage-firewall CV + blocking unit test.
5. **Step-0 behavioral existence** (UQC + Self-Eval AUROC) — cheapest kill, establishes the contrast premise.
6. Early internal-recoverability smoke test (3-fold mean)
7. Coarse layer×role sweep → freeze one primary cell.
8. Reproduce Mean Probe ≈ 0.54 + null floor + surface controls.
9. Train probe at frozen cell (leakage-safe CV, ≥5 seeds) → per-example internal scores.
10. **H4 bridge + contrast** (within-class probe↔UQC and probe↔Self-Eval) — the headline.
11. Write-up + MATS packaging.

**T1 [CP] — Pre-registration.** Freeze headline=the contrast, model, dataset, metric=AUROC, CV scheme, null band, baseline target 0.54, and every Section-F threshold, before any result is seen.
`Out:` `preregistration.md` (git-tagged, dated) · `Done:` git tag exists; file contains every downstream threshold · `Guard:` thresholds frozen before results.

**T2 [CP] — Data card + provenance + CV viability.** Load HP-KR; confirm counts (≈1,407 lies / 5,661 total); extract template-ID + persona-ID per example; use deceptive label. Resolve two things the rest of the plan assumes:

(a) Template/persona inventory. Report N_templates and N_personas and the per-template/per-persona example counts. Validate the CV scheme against them: leave-template-out needs enough templates that no single held-out fold is a large fraction of the data, and the 3-fold smoke test (T5.5) must be viable. If N_templates < 6 or N_personas < 6, the leave-X-out CV is too coarse → fall back to grouped k-fold with the available groups and record the reduced fold count in the prereg addendum.
(b) Lie provenance (load-bearing for the framing). Determine, from the dataset card/schema, which model produced the HP-KR lies. This decides what the probe measures:

If the released lies were generated by Mistral-Small-24B (or you regenerate them with it) → you are probing a model's representation of its own lie = the intended claim.
If the lies are another model's transcripts replayed through Mistral → you are probing Mistral's representation while reading another model's lie = a different, weaker claim that will be challenged in interview.
Pre-register the resolution: state explicitly which regime applies. If the dataset is cross-model and regeneration with the primary model is cheap (it is — generation is already budgeted), regenerate HP-KR lies with Mistral-24B so the probe reads first-party lies, and note this in data_card.md. If regeneration isn't faithful to the HP-KR elicitation setup, state the self-vs-other-model caveat as a known limitation and reframe the claim accordingly.



Out: data_card.md (counts, N_templates, N_personas, per-group counts, CV-viability verdict, lie-provenance regime + resolution), hpkr_with_keys.parquet ·
Done: counts match paper; grouping keys non-null for 100% of rows; N_templates/N_personas reported; CV fold count validated and recorded; lie-provenance regime stated and the first-party-vs-replayed decision committed ·
Guard: baseline reproduction (correct schema/label); CV viability checked against group counts; provenance-validity (the claim matches what the activations actually represent).

**T3 [CP] — Extraction harness.** Mistral-24B bf16 on H100; residual-stream hooks; store role-representative vectors (system / user-question / assistant-answer / post-answer) at ~10 candidate layers; **roles assigned structurally from the chat template, never from a probe.**
`Out:` `extract.py`, `acts_sweep.npz` (shape `[n,layers,roles,d]`), `token_roles.json` · `Done:` determinism round-trip identical on 10 examples; role tags verified on 10; shape matches · `Guard:` no-circularity (roles structural); storage cap C1.

**T4 [CP] — Leakage-firewall CV.** Leave-template-out + leave-persona-out splitter; unit test asserts no template/persona ID in both train and test of any fold.
`Out:` `cv_split.py`, `test_cv_leakage.py`, `cv_test.log` · `Done:` unit test PASSES (blocking) · `Guard:` leakage unit test.


**T5 [CP] — Behavioral existence (Step 0).**, staged cheap→full. Implement UQC + Self-Eval with UQC fully specified, and stage the expensive behavioral generation so the cheapest kill runs first.
UQC specification (pin before running — no agent improvisation): Pacchiardi-style Unrelated-Question Classifier. After the model produces its (potentially deceptive) answer, append a fixed, pre-registered bank of Q=12 unrelated elicitation questions (committed in preregistration.md P5), each asked in an independent continuation of the same context; collect the model's yes/no (or normalized binary token-mass) response to each. The per-example UQC score = the output of a logistic classifier over the Q-dim response vector, trained under the same leave-template/persona-out CV as the probe (a UQC trained on one set of templates/personas, scored out-of-fold on held-out ones — no leakage, mirrors the probe's discipline). This is the method that reportedly reaches 0.89–0.97 on HP-KR; if your number is far below that, the first suspect is UQC implementation, not absent signal.
Staging (fixes the "cheapest kill is actually most expensive" problem):

T5a — small-sample existence (cheap kill, ~200 balanced examples). Run UQC + the SE-A/B/C variants on ~200 examples (~100 lie / ~100 honest). Estimate UQC AUROC and the introspection-axis AUROC at low resolution. If UQC point estimate < 0.70 here, do NOT launch the full behavioral run → go straight to the Gate-2 RED branch (re-run T5a on Gemma-27B; then escalation or localization pivot). This caps wasted spend at a fraction of the ~12 GPU-hr.
T5b — full behavioral generation (only if T5a clears). Run UQC + Self-Eval on the full HP-KR set; compute final AUROCs + cluster-bootstrap CIs + the self-calibrated nulls.

Self-Eval is split exactly as before: (i) introspective-access metric = max(SE-A, SE-B), direct no-CoT readouts; (ii) SE-C = separate CoT-elicitation ceiling, never folded into the binding number. Compute self-calibrated shuffled-label nulls for both axes separately.
Out: behavioral.py, uqc_question_bank.json (the frozen Q=12), h4_step0_existence.md (T5a small-sample estimates; then T5b: UQC AUROC + CI; SE-A, SE-B, max(SE-A,SE-B), SE-C AUROCs), behavioral_scores.parquet (one column per signal/variant), selfeval_null.json (two bands) ·
Done: T5a estimates recorded and the launch/no-launch decision logged before T5b; UQC implemented per the pinned spec with the frozen question bank; UQC AUROC with cluster-bootstrap CI; SE-A, SE-B, max(SE-A,SE-B), SE-C all reported; both null bands written. Existence rules: UQC ≥ 0.70; introspective Self-Eval "at chance" iff max(SE-A,SE-B) ≤ 97.5th-pct of its null; SE-C reported separately, never in the binding number ·
Guard: pre-registered existence + staged-spend (cheap T5a gates expensive T5b) + elicitation-effort guard (strongest direct-readout SE-A/B must fail) + CoT-axis separation; self-calibrated null (guard 2); UQC-under-leakage-safe-CV (UQC scored out-of-fold, same splitter as the probe).

**T5.5 [CP] — Early internal-recoverability smoke test (cheap kill, no new GPU).** Reuse acts_sweep.npz; train single-seed L2-LR probes across ~3 leave-template-out folds (group-count-aware: if N_templates < 6 from T2, use max viable grouped k-fold) at each candidate layer; per layer take the mean OOF AUROC across folds; best-layer = argmax of that mean; route on the best-layer 3-fold mean with per-fold spread reported. (Depends on T3+T4; runs alongside/after T5.)

**T6 [CP] — Coarse sweep & freeze cell.** Train diff-of-means + L2-LR probe across the ~10×4 cells under leakage-safe CV; freeze the single primary cell to a prereg addendum **before** any bridge correlation.

## E. Five non-negotiable methodological guards

Stated as assertions/tests that must hold on every relevant commit.

1. **Leakage unit test.** `assert` no template-ID and no persona-ID appears in both train and test of any CV fold. Runs on every probe-training commit. **Blocking.**
2. **Shuffled-label null floor (extended to elicited signals; introspection axis separated from CoT axis)**. Applies to the probe (permuted-label AUROC ∈ [0.45, 0.55]) and to any elicited signal. The "Self-Eval at chance / introspectively inaccessible" claim is bound to the direct no-CoT readout metric max(SE-A, SE-B), which must fall within its own self-calibrated null band. SE-C (CoT-reasoned verdict) is a separate axis with its own null, reported alongside, never folded into the binding number — CoT measures whether the model can re-derive the lie with a scratchpad, not whether it can introspect its state.
3. **Within-class-only AND nuisance-controlled correlation**. Bridge headline = ρ_lie / ρ_honest; pooled reported only to expose the inflation gap; and a "same feature" (ρ≥0.5) claim requires ρ_lie to survive a partial correlation controlling for ans_len and ans_logprob (or a length-tercile×persona-stratified consistency check). Asserted in T9.
4. **Baseline reproduced before any lift.** Mean Probe reproduced to 0.54 ± 0.05 must exist and be committed before any recoverability / lift / bridge claim is written.
5. **Thresholds frozen before results.** `preregistration.md` (git-tagged) must predate the commit producing any gated result; the frozen primary cell is written before any bridge correlation is computed.

## F. Pre-registered decision gates

Only the gates that change what we do.

**Gate 1 — Compute go/no-go (after T0).**
Rule: C1 (≤10 GB) ∧ C2 (≤80 GB) ∧ C3 (≤40 GPU-hr / ≤$300) all PASS.
→ PASS: proceed on Mistral-24B. → FAIL C3: cut UQC question count / drop to ≥3 seeds / shrink dump. → (C2 already resolved by choosing 24B.)

**Gate 2 — The contrast existence gate (after T5/Step-0) — decides the thesis.**
🟢 **GREEN — clean contrast is the headline. UQC AUROC ≥ 0.70 AND introspective Self-Eval max(SE-A, SE-B) ≤ 97.5th-pct of its shuffled-label null (selfeval_null.json).** → directional contrast ("externally legible to UQC, not to direct self-readout SE-A/B") is the headline regardless of ρ magnitude. SE-C reported as a second axis: also at chance → strengthens ("not recoverable even with reasoning"); beats its null → sharper caveat ("reconstructable via CoT but not by direct introspection"), not a loss of headline. Proceed T5.5 → T8 → T9.
🟡 **AMBER — magnitude bridge is the headline. UQC ≥ 0.70 AND max(SE-A, SE-B) above its null band (model can introspect the lie too).** → headline = "internal direction predicts the working behavioral detector within-class on HP-KR"; report by ρ zone. SE-C still its own axis.
🔴 **RED — behavioral bridge not viable. Triggered by T5a small-sample UQC < 0.70 (cheap); re-run T5a on Gemma-3-27B; if still < 0.70**→ gated 70B-extraction escalation, else localization pivot. (Internal-side RED triggered separately by T5.5 / Gate 3.)

**Gate 3 — Recoverability / internal-direction existence (after T8).**
Rule: frozen-cell out-of-fold AUROC **≥ 0.62** (and CI lower bound above the null band).
- → YES: internal direction is real; the bridge ("internal predicts external") is meaningful → T9.
- → NO cell clears 0.62: **clean negative** ("transcript-blind lies are also near-unrecoverable internally under leakage-safe CV") — publishable; reframe the bridge as "only the external behavioral query recovers the lie," and write up accordingly.
